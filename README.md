# Stellar White Belt dApp

A simple Stellar testnet dApp built using React that allows users to connect a Freighter wallet, view their XLM balance, and send XLM transactions.

---

## Features
- Connect / Disconnect Freighter wallet  
- View XLM balance (testnet)  
- Send XLM to any address  
- Shows transaction status and hash  

---

## Tech Stack
- React  
- Stellar SDK  
- Freighter Wallet  
- Stellar Testnet (Horizon API)
